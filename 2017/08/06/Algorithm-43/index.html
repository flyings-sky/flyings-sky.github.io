字符流中第一个不重复的字符

题目描述

请实现一个函数用来找出字符流中第一个只出现一次的字符。例如：当从字符流中只读出前两个"go"时，第一个只出现一次的字符是"g"。当从该字符流中只读出前6个字符"google"时，第一个只出现一次的字符是"l"。

输出描述

如果当前字符流不存在只出现一次的字符，返回#字符

解题思路

  • 定义两个辅助数组count(int),ches(char)长度都为128(ASCII码表中最多有128个字符)，count用来记录以ch为下标的字符出现的次数，ches用来记录字符到来的顺序。
  • 在插入字符ch时，
    • 如果以当前字符ch为下标的count数组的值为0，则表示这是一个第一次出现的字符，则将它添加到ches数组中，然后将count[ch]自增.
    • 如果以当前字符ch为下标的count数组值不为0，则只将count[ch]自增即可
  • 遍历ches数组，找出第一个count值为1的字符，将其返回，找不到否则就返回#
