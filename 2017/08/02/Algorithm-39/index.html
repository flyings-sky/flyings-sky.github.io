题目描述

在一个长度为n的数组里的所有数字都在0到n-1的范围内，数组中某些数字是重复的，但不知道有几个数字是重复的。也不知道每个数字重复几次。请找出数组中任意一个重复的数字。例如：如果输入长度为7的数组{2,3,1,0,2,5,3}，那么对应的输出是第一个重复数字2

解题思路

判断数组是否为null，以及数组长度是否为0，如果是返回false

定义一个长度为length的boolean数组(默认会全部初始化为false)，然后遍历数组，每遍历到一个数字，判断：

  • 如果这个数字作为boolean数组下标的值为false，就将这个数字对应下标的boolean数组值置为true
  • 否则将这个数字存入结果数组(duplication)的下标0位置，返回true
