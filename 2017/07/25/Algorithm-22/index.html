题目描述

在数组中的两个数字，如果前面一个数字大于后面的数字，则这两个数字组成一个逆序对。输入一个数组,求出这个数组中的逆序对的总数P。并将P对1000000007取模的结果输出。 即输出P%1000000007

题目考点

归并排序的思想

解题思路

  • 将整个数列不断递归的拆分成两部分，直到每一部分长度为1，接下来一边合并相邻的子数组，一边统计逆序对的数目，将统计完成的部分从小到大进行排序，以防止在以后的统计过程中重复统计。

  • 统计逆序对思路：

    • 是用两个指针分别指向两部分子数组的末尾，并每次比较两个指针指向的数字。
    • 如果第一个子数组中的数字大于第二个子数组中的数字，则构成逆序对，并且逆序对的数目等于第二个数组中剩余数字的个数(因为两个子数组再上一步合并时已经按照递增顺序排好序)
    • 如果第一个子数组中的数字小于或等于第二个子数组中的数字，则不构成逆序对。
    • 每次比较完成后将较大的数字，从后往前复制到辅助数组中，将指针前移一位，确保辅助数组中的数字时递增排序的。
    • 整个比较完成后，使用辅助数组更新原数组，使原数组部分有序。
