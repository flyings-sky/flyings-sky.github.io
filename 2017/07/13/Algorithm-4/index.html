最大连续子序列

题目描述

给定一个数组A[0,…,n-1]，求A的连续子数组，使得子数组的和最大

例如：

数组：1,-2,3,10,-4,7,2,-5

最大子数组：3,10,-4,7,2

解法

  • 暴力法
  • 分治法
  • 动态规划法

暴力法分析

直接求解A[i,…,j]的值:

0 <= i < n

i <= j < n

数组A的长度j-i+1最大为n

因此时间复杂度O(n^3)

暴力法实现

分治法分析

  • 将数组从中间分开，那么最大子数组要么完全在左半边数组，要么完全在右半边数组，要么跨立在分界点上
  • 完全在左数组、右数组递归解决
  • 跨立在分界点上：实际上是左数组的最大后缀和右数组的最大前缀的和。因此，从分界点分别向前、后扫即可
  • 比较这三种情况，取最大的情况

分治法实现

分治法时间复杂度分析

动态规划法分析

  • 记S[i]为以A[i]结尾的数组中和最大的子数组
  • 则：S[i+1] = max(S[i]+A[i+1],A[i+1])
  • S[0] = A[0]
  • 遍历0 <= i <= n-1
  • 时间复杂度:O(n)
