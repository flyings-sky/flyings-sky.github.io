题目描述

把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。 输入一个非递减排序的数组的一个旋转，输出旋转数组的最小元素。 例如数组{3,4,5,1,2}为{1,2,3,4,5}的一个旋转，该数组的最小值为1。 NOTE：给出的所有元素都大于0，若数组大小为0，请返回0。

解题思路

由于旋转之前是一个排好序的数组，因此，这里我们可以借助二分查找的思想去实现该算法。

旋转之后，可以看成是两个排序的数组，用一个left指针来遍历第一个排序数组，用right指针来遍历第二个排序数组，则当right指针和left指针间距为1时，right指针指向的就是第二个排序数组的最小值，即整个数组的最小值。

  • 1.用left指针指向第一个数组的第一个元素，用right指针指向第二个数组的最后一个元素
  • 2.判断mid(mid = (right+left)/2)指针与left指针指向元素的大小关系
    • 如果mid值大于等于left，则mid仍在第一个排序数组中，因此更新left的值为mid
    • 如果mid值小于left，则mid在第二个排序数组中，因此更新right的值为mid
  • 3.持续更新left和right直到，left和right指针间距为1，输出right指针对应的值

两个特殊情况

  • 旋转之后和旋转之前数组相同，也就是说只存在一个非递减序列，此时数组中第一个数对应的值才是最小值
  • 如果数组中重复数字很多时，将无法明确的判断出该如何更新left和right指针，此时需要采取顺序查找的方式来查找最小值。
